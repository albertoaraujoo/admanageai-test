# Setup

## Requirements

- Node.js 20+
- npm

## Installation

```bash
npm install
```

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your values.

Main variables:

- `NANOBANANA_API_KEY`
- `AUTH_SECRET`
- `AUTH_URL`
- `AUTH_GOOGLE_ID`
- `AUTH_GOOGLE_SECRET`

## Run in Development

```bash
npm run dev
```

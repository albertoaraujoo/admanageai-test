# Features

## Home

- ad cards with hover-triggered Recreate action
- visual category filters

## Recreate

- modal with ad preview + product form
- product autofill
- image upload
- aspect ratio selection
- loading state with progress
- redirect to Projects after completion

## Products

- listing and search
- category filtering
- create, edit, and delete
- "Import from URL" flow (simulated brand fetch)

## Projects

- grid/list views
- search and filters
- per-card menu (move placeholder + delete)
- generated image viewer
